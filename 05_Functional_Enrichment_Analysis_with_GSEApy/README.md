# Functional Enrichment Analysis with GSEApy

## Overview

This project demonstrates functional enrichment analysis for single-cell RNA-seq data using GSEApy.

The workflow includes:

- Differential Expression Analysis (DEA)
- Gene Set Enrichment Analysis (GSEA)
- Leading-edge Heatmap
- Preranked GSEA
- Enrichr Over-Representation Analysis (ORA)
- Enrichment Dotplot
- Enrichment Network Visualization

---

## Workflow

1. Load AnnData object
2. Select cell type
3. Differential expression analysis
4. Run GSEA
5. Visualize leading-edge genes
6. Perform preranked GSEA
7. Run Enrichr analysis
8. Visualize enrichment results
9. Build enrichment network
