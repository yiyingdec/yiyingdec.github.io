# Basic Single Cell Workflow

This repository documents my learning journey in single-cell RNA sequencing (scRNA-seq) analysis using Python and Scanpy.

## Objectives

- Learn the standard single-cell RNA-seq analysis workflow.
- Understand the biological meaning behind each analytical step.
- Build reproducible computational biology projects.

---

## Workflow

1. Quality Control (QC)
2. Cell Filtering
3. Gene Filtering
4. Normalization
5. Highly Variable Genes (HVG)
6. Principal Component Analysis (PCA)
7. Neighbor Graph Construction
8. UMAP Visualization
9. Leiden Clustering
10. Marker Gene Identification
11. Differential Expression Analysis

---

## Software

- Python
- Scanpy
- Anndata
- Matplotlib
- Pandas
- NumPy
